🧩 Exercício sobre Classes Abstratas

📚 Sobre o Projeto

Exercício desenvolvido em sala, sob orientação do professor Leandro Procopio Alves, da disciplina Programação de Soluções Computacionais (Ciência da Computação – Universidade Anhembi Morumbi, campus Mooca),
com foco na compreensão de classes abstratas em Java.

🎯 Objetivo

Explorar o uso de classes abstratas em Java, permitindo:

Definir estruturas base genéricas

Criar métodos abstratos que devem ser implementados pelas subclasses

Reforçar conceitos de herança e polimorfismo

🧩 Conceitos Trabalhados

Durante o desenvolvimento, foram abordados:

🧱 Classes abstratas (abstract)

🔁 Herança

🎭 Polimorfismo

🧠 Sobrescrita de métodos (@Override)

🧠 Arquitetura do Sistema

Neste projeto, utilizei os pilares da POO para resolver um problema clássico de geometria de forma escalável. A ideia central foi criar um 'contrato' através de uma classe abstrata, garantindo que qualquer figura — seja um círculo, quadrado ou triângulo — saiba exatamente como calcular sua própria área. O uso de polimorfismo permitiu que o sistema trate diferentes formas de maneira uniforme, tornando o código muito mais limpo e preparado para a adição de novas figuras sem a necessidade de reescrever a base do projeto.

O programa demonstra como uma classe abstrata pode servir como base para outras classes, garantindo que determinados métodos sejam obrigatoriamente implementados, promovendo maior organização e reutilização de código.

Diferença entre classes concretas e abstratas

Importância da abstração no desenvolvimento de códigos

Reutilização e organização de código

Estrutura hierárquica entre classes

🧾 Conclusão

A realização deste exercício me permitiu compreender na prática o papel das classes abstratas na Programação Orientada a Objetos.

Esse conceito é fundamental para a criação de sistemas mais organizados, reutilizáveis e escaláveis, pois permite definir comportamentos comuns que serão compartilhados e especializados pelas subclasses.
